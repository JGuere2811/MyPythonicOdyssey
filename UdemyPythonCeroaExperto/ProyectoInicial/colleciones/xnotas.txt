#colecciones en python
 una coleccion es un conjunto de datos, en python tenemos varios tipo
 que podemos utilizar con el objetivo de almacenar, organizar y manipula
 multriples conjuntos de datos, por ello tambien se les conoce como
 estructura de datos.
 tipos a estudiar:
 listas
 tuplas
 set(conjunto)
 diccionarios
 estos son los tipos mas comunes y mas utilizados. comencemos con el tipo
 listas
 ####lista en python###
 Las listas son colecciones ordenadas y mutables de elementos que pueden
 ser de diferentes tipos. las listas son dinamicas, lo que significa que
 cambiar de tamaño, pódemos añadir, modificar o eliminar elementos.

#sintaxis listas
mi_lista = [elemento1, elemento2, elemento3]


#Ejemplo de lista
numeros = [1,2,3,4,5]
frutas = ["manzanas","banana","cereza"]
mixta = [1,"dos",3.0,[4, 5]]

####tuplas en python
las tuplas son similares a las listas, ya que tambien son una coleccion de
datos ordenados, pero las tuplas son inmitables, lo que significa que una 
vez  "CREADA NO ES POSIBLE MODIFICAR SU TAMAÑO, NI PODEMOS AGREGAR MAS
ELEMENTOS NI ELIMINARLOS".
la tupla suelen utilizarse para crear colecciones de datos que no deben
cambiar con el tiempo
#sintaxis de una tupla
m1_tupla = (elemento1,elemento2,elemento3)
m1_tupla_sin_parentesis = elemento1,elemento2,elemento3
#Ejemplo de tuplas 
tupla_numeros = (1, 2, 3, 4, 5)
tupla_mixta = ("manzana", 10, 3.14, [1, 2, 3])
m1_tupla_sin_parentesis = 'juan', 'karla'
tupla_un_elemento, = 10 #La coma no es opcional
###Sets en pythonlos sets(conjuntos) son colecciones de datos no ordenada

haya elementos duplicados en nuestra coleccion
#Sintaxis de un set
mi_set = {elemento1, elemento2, elemento3}

#Ejemplo de sets
set_a = {1, 2, 3, 4}

#Diccionarios en python
Los diccionarios son una coleccion de datos ordenada (3.7+) y se almacenar
en forma de llave:valor. Es una estructura muy util cuando necesitas 
asociar un conjunto de  valores con un conjunto de claves o llaves
que sirven como indices unicos
#sintaxis de un diccionario
mi_diccionario = {clave1: valor1, clave2: valor2}

#Ejemplo o de un diccionarios
personas = {'nombre': 'Pedro', 'edad': 30, 'es_cansado': True}
